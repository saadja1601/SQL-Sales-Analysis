# Sales Data Analysis (SQL Project)

## Objective
Analyze sales data to uncover insights related to revenue, customer behavior, product performance, and regional trends.

## Dataset
Sales dataset containing information on transactions, products, customers, pricing, and regions.

## Tools Used
- SQL
- Excel

## Key Analysis Performed
- Revenue analysis by region and product category
- Sales performance by sales representatives
- Monthly sales trend analysis
- Customer type segmentation (New vs Returning)
- Payment method usage analysis
- Profit estimation using cost and price data

## Key Insights
- Certain regions contribute significantly higher revenue than others
- A small number of sales representatives generate the majority of sales
- Product categories vary in profitability, not just revenue
- Returning customers contribute a consistent portion of revenue
- Discounts vary across product categories and impact margins
